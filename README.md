# 🌍 Disaster-O-Meter

AI-Based Multi-Disaster Damage Severity Prediction System

Disaster-O-Meter is an intelligent disaster assessment platform designed to classify and predict damage severity across multiple disaster categories including floods, earthquakes, storms, and landslides.

The system combines traditional computer vision techniques with deep learning models to provide rapid and accurate disaster impact analysis for emergency response teams and decision-makers.

## Key Features

* Multi-Disaster Classification
* Damage Severity Prediction
* OpenCV Feature Extraction
* VGG16 Deep Learning Architecture
* Real-Time Inference
* Edge Device Deployment
* Drone Integration Support
* Emergency Response Analytics

## Supported Disaster Types

* Floods
* Earthquakes
* Landslides
* Storms

## Severity Levels

* No Damage
* Mild Damage
* Severe Damage
